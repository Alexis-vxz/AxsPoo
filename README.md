## 📌 Descripción del Repositorio
Este repositorio contiene las prácticas y ejercicios desarrollados durante el curso de **Programación Orientada a Objetos**. Aquí se exploran conceptos fundamentales como:

* **Clases Abstractas e Interfaces**: Implementación de moldes base para mobs y herramientas.
* **Herencia y Polimorfismo**: Especialización de clases (Vaca, Creeper, Arco, etc.).
* **Manejo de Excepciones**: Validaciones robustas y creación de excepciones personalizadas.
* **Persistencia de Datos**: Registro de información en archivos externos `.txt`.

---

## 🚀 Proyectos Destacados

1.  **Sistema de Mobs (Minecraft Style):** Uso de herencia para crear criaturas con comportamientos únicos.
2.  **Gestión de Herramientas:** Implementación de durabilidad y lógicas de combate (incluyendo el sistema de flechas para el Arco).
3.  **Registro de Calificaciones:** Programa con manejo de errores (`try-except-finally`) y guardado de datos.
